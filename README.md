- css폴더와 파일을 따로만들어 html에 연결해야한다.
- 각 박스 간의 간격은 margin값은 20px 밀어야 한다.
- 각 박스들은 width, height를 지정하지 못하고 padding으로만 크기를 잡아야한다.
- 글자 크기는 70px이어야 한다.
